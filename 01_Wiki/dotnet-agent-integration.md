> [!IMPORTANT]

## Core Concepts
- **Semantic Kernel Bridge**: Using Microsoft's Semantic Kernel to expose .NET methods as agent tools.
- **MCP for .NET**: Implementation patterns using the C# MCP SDK.
- **Dependency Injection for Agents**: Patterns for injecting agentic capabilities into standard ASP.NET Core or WPF applications.
