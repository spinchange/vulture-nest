# PKM History MOC

This map traces the conceptual lineage behind the vault's design: from early visions of associative knowledge to collaborative augmentation and modern shared-memory systems. It is less a chronology of software products than a map of the ideas that explain why the Nest is structured the way it is.

## The Visionaries
Read these in order if you want the cleanest intellectual path into the cluster.

* [[memex]]: Vannevar Bush's 1945 vision of associative linking.
* [[augmenting-human-intellect]]: Douglas Engelbart's framework for boosting human capability.
* [[collective-iq]]: The goal of using networked knowledge to solve complex problems.

## Narrative Through-Line

The cluster has a clear progression:

1. [[memex]] defines the idea of associative trails rather than hierarchical filing.
2. [[augmenting-human-intellect]] reframes tools as part of a larger human-language-artifact-method system.
3. [[collective-iq]] scales augmentation from the individual to the organization.

That progression is the historical substrate under [[llm-wiki-pattern]], [[wiki-as-codebase]], and the vault's multi-agent collaboration model.

## Where To Start

- If you want the origin point, start with [[memex]].
- If you want the strongest conceptual bridge into modern agent tooling, start with [[augmenting-human-intellect]].
- If your question is organizational knowledge rather than individual tooling, start with [[collective-iq]].
