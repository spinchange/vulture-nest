# Core Patterns MOC

This hub defines the architectural and philosophical standards of this vault, specifically the **YANP (Yet Another Note Protocol)**.

## Status of Standards
This MOC mixes adopted operating conventions with more exploratory architectural concepts. Treat it as a map of the vault's design language, not as a claim that every linked pattern is equally mature or fully implemented.
