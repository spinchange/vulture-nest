# Plain+ Design

**Plain+** is a design framework created to counter the ubiquitous "AI fingerprint" in modern software interfaces. It focuses on human-centered, editorial design through extreme restraint and semantic clarity.

## Core Philosophy
The framework is built on the [[anti-ai-aesthetic]], which rejects common LLM-generated design tropes in favor of structural and typographic discipline.

## Foundational Rules
- **No Border-Radius**: All UI elements are square.
- **No Gradients or Shadows**: Depth is achieved via spacing and hierarchy.
- **Warm Neutrals**: Uses a #1a1a1a on #fdfdfc color scheme to mimic physical paper.
- **Typographic Contrast**: Pairs Serif body text with Sans-Serif headings.

## Application in Vulture Nest
As a [[the-compounding-artifact]], this wiki should strive to embody Plain+ principles in its presentation, ensuring that the interface remains transparent and the content remains the focus.
