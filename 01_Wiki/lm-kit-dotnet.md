# LM-Kit.NET

**LM-Kit.NET** is a high-performance SDK designed for integrating Large Language Models (LLMs) directly into .NET applications. It focuses on local execution and hardware acceleration.

## Key Features
- **Local Inference:** Run models entirely on-device without cloud dependencies.
- **Hardware Acceleration:** Optimized for CUDA, Metal, and DirectCompute.
- **Multi-Framework:** Compatible with WPF, ASP.NET Core, and Console apps.

## Agentic Integration
LM-Kit provides the "brains" for .NET-based agents, allowing them to process natural language and generate responses while maintaining data privacy.
