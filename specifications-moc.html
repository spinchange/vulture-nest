Specifications & RFCs MOC

This map organizes the formal specifications, Requests for Comments (RFCs), and technical protocols that define the architecture and behavior of this vault and its agentic systems.

📜 Vault Protocols (YANP)

  • yanp-for-agentic-workflows - The core note protocol for the vulture-nest.
  • yaml-for-yanp - Metadata and frontmatter standards.
  • yanp-frontmatter - Detailed specification for opening YAML blocks.
  • frontmatter-schema-rfc-2026-05-04 - Active RFC for refreshing the frontmatter schema.
  • executable-note-standard - Proposal for active knowledge and embedded logic.

🛰️ Inter-Agent Protocols

  • a2a-protocol - The foundational Agent-to-Agent protocol.
  • a2a-capability-lattice - Formalizing agent capabilities and discovery.
  • capability-lattice-spec - Technical specification for the lattice.
  • a2a-mcp-contrast - Comparing A2A with the Model Context Protocol.
  • inter-agent-handoff-protocol - Rules for session state transfer.
  • artifact-write-protocol-rfc-2026-05-03 - Active RFC for formalizing vault writes.
  • note-hardening-pattern-rfc-2026-05-03 - Active RFC for graduating fleeting notes.

🤖 Orchestration & Automation

  • spec-hermes-agent-loop - Descriptive spec for Hermes turn execution, tool dispatch, message alternation, compression, and persistence.
  • spec-agentic-source-orchestrator - The "Knowledge Compiler" v1.
  • spec-agentic-source-orchestrator-v2 - Architecture for automated ingestion.
  • spec-agentic-source-orchestrator-v3 - Advanced ingestion and synthesis pipeline.
  • knowledge-compiler-spec - Detailed requirements for vault-as-codebase tools.
  • spec-memory-mcp - Implementation spec for the long-term memory server.
  • spec-visual-vault-language - Formalizing the visual representation of the graph.
  • spec-knowledge-gardening - Operational standards for vault maintenance.
  • spec-chatgpt-web-mcp-wrapper - Architecture for the MCP bridge.
  • spec-firecrawl-pgvector-pipeline - Ingestion pipeline for high-fidelity web scraping.

🧪 Experimental Standards

  • protocol-source-ingestion-runbook - Operational guide for adding new sources.
  • experiment-capture-protocol - Standard for recording vault experiments.
  • polyglot-adr-rfc - Architectural Decision Record for the polyglot platform.
  • polyglot-platform-adr - Finalized ADR for multi-language agent support.
